# Лабораторные работы по ETL

Этот репозиторий содержит материалы и инструкции по выполнению лабораторных работ по предмету **ETL (Extract, Transform, Load)**. Здесь представлены шаги по настройке инструментов, созданию конвейеров данных, интеграции нескольких источников и анализу данных с помощью Dask и Airflow.

## Содержание лабораторных работ

1. **[Установка и настройка ETL-инструмента. Создание конвейеров данных](https://github.com/KharlamovaAnn/ETL/blob/main/Lab_1/README.md)**  
   Основы работы с ETL-инструментами, создание простых конвейеров, запуск первых задач.

2. **[Динамические соединения с базами данных]()**  
   Настройка подключения к различным базам данных, использование переменных и параметров для гибкой работы с источниками данных.

3. **[Интеграция данных из нескольких источников. Обработка и согласование данных]()**  
   Объединение данных из разных источников, обработка и нормализация данных для дальнейшего анализа.

4. **[Анализ данных с помощью Dask и визуализация графов (DAG)]()**  
   Использование Dask для параллельной обработки данных и визуализация DAG для отслеживания выполнения задач.

5. **Создание ETL-конвейеров:**

   > - **[Проектирование объектной модели данных. Проектирование сквозного конвейера ETL]()**  
   >   Определение структуры данных, создание схемы и планирование полного конвейера ETL.  

   > - **[Разработка алгоритмов для трансформации данных. Airflow DAG]()**  
   >   Разработка логики обработки данных, написание DAG для автоматизации задач и управления зависимостями.  

6. **[Разработка полного ETL-процесса. Оркестровка конвейера данных]()**  
   Объединение всех этапов в единый процесс, настройка расписания выполнения и мониторинг работы конвейера.
